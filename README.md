# Todo App

A modern and responsive Full Stack Todo Application built to manage daily tasks efficiently. Users can create, view, update, and delete todos with a clean and user-friendly interface.

## 🚀 Features

- Add new tasks
- View all todos
- Edit existing tasks
- Delete tasks
- Mark tasks as completed
- Responsive UI
- REST API integration
- Real-time updates

## 🛠️ Tech Stack

### Frontend
- React.js
- Axios
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose
